AIM:
  To write a JavaScript program to change the colour of a web page to the colour typed by the user in the text box.
ALGORITHM:
    1.	Start the notepad and write HTML document with <html> tag.
    2.	Create a head section using <head> tag and set a title using the <title> tag.
    3.	Start the body section using <body> tag.
    4.  Add a heading (<h1>), a text input for the color, and a "Change Color" button.
    5.  Write a JavaScript function to get the input value and change document.body.style.backgroundColor.
    6.  Link the button to the function using the onclick attribute.
    7.	Connect the buttonâ€™s onclick attribute to the JavaScript function.
    8.	Save the file with .html extension and open it in a browser to test.
PROGRAM:
    <! DOCTYPE html>
    <html lang="en">
    <head>
    <meta charset="UTF-8">
    <title>Change Background Colour</title>
    </head>
    <body>
    <h1>Change Background Colour</h1>
    <input type="text" id="colourInput" placeholder="Enter a color or hex code (e.g., red or #ff0000)">
    <button onclick="changeColour()">Change Colour</button>
    <script>
    function changeColour() {
    const color = document.getElementById('colourInput').value;
    document.body.style.backgroundColor = color;
    }
    </script>
    </body>
    </html>
RESULT:
      Thus the above HTML Code is executed and the output is obtained.
